# 8Knot Openshift Configuration

This repo contains some openshift configuration that was previously used for deploying 8knot on openshift

It is no longer being used, deployed, or maintained in this form, but may be useful as a starting point or sample configuration that may help someone else who wishes to deploy 8Knot on openshift.

All "secrets" in this repo are placeholder/sample values.



## History

This repository was created by extracting the git history of the `openshift` directory of the 8Knot repository prior to its deletion from the main 8Knot repository. The LICENSE.md was preserved during this process.

All commits prior to the introduction of this README are essentially copies of their respective commits in the 8Knot repository.
